Что такое CSS?
Стилем или CSS (Cascading Style Sheets, каскадные таблицы стилей) называется набор параметров форматирования, который применяется к элементам документа, чтобы изменить их внешний вид. Возможность работы со стилями издавна включают в развитые издательские системы и текстовые редакторы, тем самым позволяя одним нажатием кнопки придать тексту заданный, заранее установленный вид. Теперь это доступно и создателям сайта, когда цвет, размеры текста и другие параметры хранятся в определенном месте и легко «прикручиваются» к любому тегу. Еще одним преимуществом стилей является то, что они предлагают намного больше возможностей для форматирования, чем обычный HTML.
CSS представляет собой мощную систему, расширяющую возможности дизайна и верстки веб-страниц.

Введение.
Если вы впервые решили создать стиль для "Неофициального клиента 4PDA", вам пригодится основательное понимание селекторов CSS.
Селектор это инструкция в наборе правил CSS, которая указывает браузеру(в нашем случае клиенту 4PDA) какой элемент выбрать для применения к нему стиля.
Универсальный селектор
Данный селектор состоит из одной звездочки:
* {
background-color: red;
}

Использование в таком виде означает выбор всех элементов на странице.

Селектор типа элемента
Также известный как «селектор типа», данный селектор выбирает элемент HTML по названию тега.
h2 {
background-color: red;
}

div {
background-color: red;
}


Селектор класса
Выбирает элемент с именем класса, указанным в атрибуте класса в коде HTML.
.element {
background-color: red;
}

Можно указывать несколько разделенных пробелами имен классов в одном атрибуте класса, что делает этот селектор очень мощным инструментом.

Селектор идентификатора
Данный селектор выбирает элемент с идентификатором, указанным в атрибуте id в HTML.
#element{
background-color: red;
}


Селектор потомка
Данный селектор задается пробелом, разделяющим два селектора, и выбирает дочерний элемент, не обязательно непосредственный, но также на более глубоких уровнях вложенности.
.parent .child {
background-color: red;
}


Селектор атрибута
Данный селектор выбирает элемент, основываясь на атрибуте HTML или/и его значении.
Пример №1
div[style] {
background-color: red;
}

Пример №2
input[type="text"] {
background-color: red;
}

В первом примере выбирается элемент <div>, у которого есть атрибут «style». Во втором примере выбирается элемент <input>, у которого есть атрибут «type» со значением «text».

Селектор дочернего элемента
Такой селектор выбирает непосредственный дочерний элемент заданного элемента:
.one > .two {
background-color: red;
}

Стиль не будет применяться к элементу «.two», если он не является непосредственным дочерним элементом «.one». Не применяется к элементам с большим уровнем вложенности. Затрагивает только сам дочерний элемент, не родителя.

Селектор смежного одноуровневого элемента
Этот селектор использует знак "+" и выбирает элементы смежные друг другу или непосредственные одноуровневые элементы, в обоих случаях элементы должны иметь одного родителя.
h2+p {
background-color: red;
}


Селектор общего одноуровневого элемента
Использует тильду (~) и работает так же как и селектор смежного сестринского элемента, за исключением того, что элементы не обязаны быть непосредственными элементами одного уровня.
a[data-info~="external"] {
color: red;
}
a[data-info~="image"] {
border: 1px solid black;
}


Псевдокласс
Технически относясь к категории селекторов, псевдоклассы обычно не называются селекторами. Они выбирают элемент по тому состоянию, в котором данный элемент находится. Несколько примеров:
Пример №1
a:visited {
background-color: red;
}

Пример №2
input:focus {
background-color: red;
}


Псевдоэлемент
И снова тип, который обычно не называют селектором. Псевдоэлементы обычно представляют собой элементы страницы HTML, которые не являются частью обработанного кода HTML:
p:first-letter {
background-color: red;
}

p:before {
content: "";
background-color: red;
}

Псевдоэлементы «:before» и «:after» уникальны на фоне других тем, что должны использоваться совместно со свойством content, но это уже совсем другая история! :)


Урок №1. Фон
Пример-1
Допустим нам нужен красный цвет фона нашего стиля. Это можно реализовать при помощи CSS-кода:
body {
background-color: #FF0000;
}

Этот пример демонстрирует фундаментальную модель CSS:
selector{property: value;}

selector (селектор) - К какому HTML тэгу применяется свойство (в нашем примере "body")

property(свойство) - Свойство, которое, к примеру, может быть цветом фона ("background-color")

value(значение) - Значением свойства background-color может быть, например red ("#FF0000")

Пример-2
Устанавливаем картинку вместо цвета. Я использовал картинку размером 28х28 и сделал ее повторяющейся по горизонтали и по вертикали:
HTML body{
background: url('bg.png') repeat #fff;
}

url('bg.png') - путь к моей картинке. В моем случае картинка лежит в одной папке со стилем (*.css).
repeat - Картинка повторяется по горизонтали и по вертикали.
#fff - цвет фона.

Урок №2. Шрифт
@font-face. Нестандартный шрифт средствами CSS.

Нестандартный шрифт в отличии от стандартных — это шрифт, который с большой вероятностью будет отсутствовать у большинства пользователей клиента.
Задача: Внедрить нестандартный шрифт правилом @font-face.
CSS правило @font-face позволяет импортировать внешний файл шрифта в документ.
У меня имеется шрифт DroidSans.ttf.
Шрифт DroidSans.ttf


Для начала я закидываю его в папку рядом со стилем который будет использовать данный шрифт. У меня это папка /data/4pdaClient/styles/black. Теперь редактируем стиль. Я хочу установить шрифт для всего стиля, а не для отдельного элемента.
Для начала нам нужно объявить шрифт. И так, пишем:
@font-face {
font-family: 'Droid Sans'; /* имя шрифта для CSS правил */
src: local('Droid Sans'), /* проверяем наличие шрифта на устройстве */
url('DroidSans.ttf') format('ttf'), /* формат шрифта */
url('DroidSans.ttf') format('TrueType')
url('DroidSans.ttf') format('OpenType');
src: url('DroidSans.ttf'); /* путь до шрифта */
font-weight: normal;
font-style: normal;
}

Теперь нужно применить этот шрифт. Добавляем:
HTML body {
font-family: 'Droid Sans',Verdana,Tahoma,Arial,'Trebuchet MS',sans-serif,Georgia,Courier,'Times New Roman',serif;
}
